projects

MOCAP Animation
in a team of 3, i helped to create a tool that allows users to make animations in the browser. using the cmu mocap database, we wrote parsers for two different types of files and created an online gui that translated the data into trajectories and rendered them in javascript. this was my final project for cs290: digital 3d geometry with chris tralie.

technologies: javascript, webgl, html, css
maddiestambler.com
last summer, i made a website for a fellow duke student. she is a photographer, film maker, and most importantly, a storyteller. it was one of my first times working with web development and through it, i was able to learn a ton! be sure to check out her amazing work!

technologies: html, css, javascript, jquery
VOOGASalad
in a team of 12, i helped create a game engine and authoring environment that allows users to easily create new tower defense games. the project took about two months and was the final assignment for cs308: software design and implementation. the course emphasized clean, efficient code as well as scalable design.

technologies: java, javafx
Space Chase
my first "big" java project. i made a simple scroller game using javafx. i cringe when i look back at the code now but it was my first big project alone! gotta start somewhere, right?

technologies: java, javafx
