VOOGASalad

Game engine.

TEAM SIZE: 12
TECHNOLOGIES: Java, JavaFX, XML

ABOUT:
VOOGASalad stands for Video Object Oriented Game Architecture Salad (based on GameSalad, a mobile game engine). It is the largest project I have worked on and was about a month and a half long final assignment in my software class. We designed a game engine, game authoring envirionment, and game player. By the end, our program allowed users to build and customize their own tower defense games and simple RPGs. From there, the engine and player were able to load and allow users to play any of the created games.

SLogo

IDE for Simple Logo.

TEAM SIZE: 4
TECHNOLOGIES: Java, JavaFX, XML

ABOUT:
SLogo stands for Simple Logo. This was one of my first software projects to involve a team. In a group of four, we designed and implemented an IDE. It was able to control a turtle on the screen via commands such as FORWARD, BACK, etc. Moreover, the IDE was capable of handling for loops and basic mathematic computations.
